# VoiceAI-Embedded

An ESP32-S3 voice AI embedded sample based on ESP-IDF. Under `examples/espressif/esp32s3/` you will find the `esp32s3_webrtc` (WebRTC) and `esp32s3_ws` (WebSocket) projects, with audio codecs, display, and LEDs integrated. After you have set up ESP-IDF, enter the corresponding subdirectory and build and flash with `idf.py`.
